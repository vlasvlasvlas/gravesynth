YAML de síntesis

El YAML define el sintetizador de ese portal. Al editarlo, el sonido cambia en vivo si el texto es válido.

  • type elige el motor: MonoSynth, FMSynth o AMSynth.
  • oscillator.type cambia la onda: sine, square, triangle o sawtooth.
  • envelope controla ataque, caída, sustain y release.
  • filter, harmonicity y modulationIndex cambian color, brillo y carácter.
type: "MonoSynth"
oscillator:
  type: "sawtooth"
envelope:
  attack: 0.01
  decay: 0.2
  sustain: 0.5
  release: 1.2

Usa dos espacios para indentar. Si aparece error, corrige el YAML y el sonido anterior queda activo.
